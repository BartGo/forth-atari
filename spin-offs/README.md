# Study-oriented fig-Forth modification for 8-bit Atari

How to run:
* Use Atari 8-bit emulator like Altirra with Atari XL ROM.
* Boot IDE-BOOT.ATR 
* Enjoy?

Features:

1. Informative banner after booting
   
![banner after booting](../drafts/s1boot.png)

2. Built-in help on the most often used "words"

![built-in help](../drafts/s2help.png)

3. Word header viewer and decompiler in one convenient "/?" command

![word header and decomp](../drafts/s3head.png)

4. Available a full-screen editor published in "Micro 6502", with slight modifications (including help)

![editor](../drafts/s4edit.png)

![editor](../drafts/s5edhelp.png)
